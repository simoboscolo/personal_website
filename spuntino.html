Spuntino

Share fast recipes with the community.

Spuntino has been developed using Ruby on Rails for my final project at the New York Code+Design Academy. It allows users to share short-preparation recipes, upload pictures and comment them.

The streght of this project is the extremely simple design leading to a great User Experience.

You can have a look a it from here. As it was a school project some functionalities had never been ended. As now, you are able to register, create and edit recipes or comment.

Everything unneccessary has been discarded in order to achieve a minimal and simple design led to improve the user experience.

The recipe picture appear as a background and loses importance while scrolling, so user can focus on preparation and ingredients.
